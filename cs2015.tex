Hash function = hash code (keys to integer) + compression function (integers to $\llbracket0, N-1\rrbracket$)

                @{}p{\linewidth-\the\MyLen}@{}}
Memory address & Good except for numeric and string keys \\
Integer cast & Reinterpret the bits as an integer \\
Component sum & Partition the bits into components + sum the components \\
Polynomial\newline accumulation   & $p(z) = a_0 + a_1z +...+a_{n-1} z^{n-1}$ où $z$ est fixé et $a_i$ sont les components.
\end{tabular}
